PDF Plot Generation from TLO Manifest

Problem

At present there is limited oversight to train loading. Data is currently collected for every wagon loaded but it is rarely used and never visualised. This causes loading inefficiencies to occur.

Solution

Create a python script to interpret the manifest for train loading and auto-generate a PDF with plots of relevant train loading data. Have this emailed out to stakeholders after the completion of every train.

By providing timely insights, the generated performace plots enhance oversight and transparency, helping teams identify and address operational issues. The visualisations make it easy to spot abnormal loading scenarios and detect excessive manual operations. They also help pinpoint incorrect manual slider settings and facilitate bench identification during overload events

Creation from PlantSCADA

After a train is completed, a cicode function is triggered. This creates a .txt file with every wagon's loading data.

Files Needed for Processing and Emailing

Python script packaged as an executable is called. This processes the latest txt file and outputs the PDF plots. Next, a batch file for emailing the PDF manifest to recipients is triggered.
